# GITHUB CLI CHEAT SHEET  
### GITHUB PULL REQUEST  
- gh pr status
- gh pr list
- gh pr list -s all
- gh pr view ['number' | 'url' | 'branch']
- gh pr checkout
- gh pr create -t 'title' -b 'body'
### GITHUB ISSUES
- gh issue status
- gh issue list
- gh issue list -s all
- gh view ['number' | 'url']
- gh issue create -t 'title' -b 'body'
### GITHUB REPOSITORIES
- gh repo clone 'repo'
- gh repo create 'name repo'
- gh repo fork 'repo'
- gh repo view 'repo'
